Dockerfile Example:

# Use Ubuntu as base image
FROM ubuntu:20.04
# Prevent interactive prompts
ENV DEBIAN_FRONTEND=noninteractive
# Install NGINX
RUN apt update && apt install -y nginx
# Copy custom HTML file into the container
COPY index.html /var/www/html/index.html
# Start nginx in foreground
CMD ["nginx", "-g", "daemon off;"]

Also create an index.html file in the same folder:
<h1>Hello from custom NGINX container!</h1>

mkdir custom-nginx && cd custom-nginx
docker build -t my-nginx-image .
docker run -d -p 8080:80 my-nginx-image
http://localhost:8080


Volume Hands-On (Named Volume)

docker volume create mydata
docker run -d -p 8081:80 -v mydata:/usr/share/nginx/html nginx
docker cp index.html <container_id>:/usr/share/nginx/html/index.html
http://localhost:8081

Bind Mount (Host Folder Mounting) 

docker run -d -p 8084:80 -v $(pwd):/usr/share/nginx/html nginx
